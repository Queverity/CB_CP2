
# Sierpinski Triangle Fractal Generator
This is a program that allows the user to generator a sierpinski triangle of a specified fractal depth and color.

## How To Use
1. Pip install the turtle2img module
2. Run the fractal_generator.py file

## Project Features
- Generate a Sierpinski Triangle
- Choose color and fractal depth of sierpinski triangle
- Save the turtle generated triangle as a PNG file
